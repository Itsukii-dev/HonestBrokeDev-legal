Privacy Policy

Ultimo aggiornamento: aprile 2025 · Versione app: 1.0.0

1. Titolare del trattamento

Il titolare del trattamento dei dati è Emanuele Pina, sviluppatore indipendente.
Contatto: itsukii-dev@outlook.it

2. Cosa raccogliamo e perché

L'app raccoglie un numero minimo di dati, tutti necessari al funzionamento:

Dato Come viene generato Dove viene conservato Scopo Visibile ad altri utenti?
Nickname Inserito dall'utente al primo avvio Supabase (EU) + dispositivo locale Personalizzazione e contatore pubblico dei download Sì — appare nel ticker live dell'app
Device ID UUID generato casualmente dall'app al primo avvio, non collegato all'Apple ID né all'hardware Supabase (EU) + dispositivo locale (AsyncStorage) Evitare il conteggio di download duplicati No
Push Token Fornito dal sistema Expo/APNS solo se l'utente concede il permesso Supabase (EU) Inviare notifiche sui traguardi di download (milestone) No

Non raccogliamo: posizione geografica, dati sanitari, dati finanziari, contatti, cronologia di navigazione, dati biometrici, IDFA o qualsiasi identificatore pubblicitario.

3. Nickname pubblico — avviso importante

Il nickname inserito al primo avvio viene mostrato pubblicamente nel ticker live visibile a tutti gli utenti dell'app, insieme al numero progressivo di download. Non inserire dati personali reali (nome completo, email, ecc.) come nickname.

4. Base giuridica del trattamento

Il trattamento si basa su:

  • Esecuzione del contratto (Art. 6.1.b GDPR): il nickname e il Device ID sono necessari per fornire il servizio (assegnare un numero progressivo univoco).
  • Consenso (Art. 6.1.a GDPR): il push token viene raccolto solo dopo esplicita autorizzazione dell'utente tramite il sistema di permessi iOS.

5. Conservazione dei dati

I dati vengono conservati per tutta la durata dell'app. In caso di chiusura del servizio, i dati verranno eliminati entro 30 giorni. Su richiesta, i dati relativi a uno specifico Device ID possono essere cancellati prima.

6. Condivisione con terze parti

I dati non vengono venduti né ceduti a scopo pubblicitario. Vengono condivisi solo con:

  • Supabase Inc. — infrastruttura database, server nella regione EU. Privacy Policy Supabase
  • Expo / Expo Push API — sistema di notifiche push. I push token vengono trasmessi ai server Expo per la consegna delle notifiche. Privacy Policy Expo

Il link a BuyMeACoffee presente nell'app apre un sito esterno: Honest Broke Dev non trasmette nessun dato a BuyMeACoffee.

7. Trasferimento internazionale

Supabase e Expo sono soggetti al GDPR e operano in conformità con le normative europee sulla protezione dei dati. I dati vengono archiviati su server nell'Unione Europea (Supabase EU region).

8. I tuoi diritti (GDPR)

In qualità di interessato, hai il diritto di:

  • Accedere ai dati che ti riguardano
  • Richiedere la rettifica di dati inesatti
  • Richiedere la cancellazione dei tuoi dati (diritto all'oblio)
  • Opporti al trattamento
  • Revocare il consenso in qualsiasi momento (es. revocando i permessi notifiche in Impostazioni iOS)

Per esercitare questi diritti, scrivi a itsukii-dev@outlook.it indicando il tuo nickname e/o Device ID (visibile nelle impostazioni del dispositivo tramite AsyncStorage). Risponderemo entro 30 giorni.

9. Sicurezza

I dati sono conservati su Supabase con Row Level Security (RLS) abilitata. Il Device ID non è collegato ad alcun identificatore hardware o all'Apple ID. Le connessioni tra l'app e il server avvengono esclusivamente tramite HTTPS/TLS.

10. Minori

L'app è classificata 4+ e non richiede registrazione. Non è progettata per raccogliere intenzionalmente dati di minori di 13 anni. Se sei un genitore e ritieni che tuo figlio abbia fornito dati personali, contattaci per la cancellazione immediata.

11. Modifiche a questa informativa

Eventuali modifiche verranno pubblicate su questa pagina con data di aggiornamento. Per modifiche sostanziali, gli utenti attivi potranno essere notificati tramite le notifiche push (se il permesso è stato concesso).
